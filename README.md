# ABM Creator – Dashboard

Dashboard para criação e gestão de carrosséis para redes sociais com integração ao Canva e Instagram.

## Funcionalidades

- Criação de carrosséis com IA (Gemini API)
- Preview interativo de slides
- Edição de slides com modal
- Integração com Instagram

## Como rodar

```bash
cd server
npm install
cp .env.example .env
# Preencha as variáveis no .env
npm start
```

Acesse `http://localhost:3000`

## Variáveis de ambiente

Veja o arquivo `.env.example` para as variáveis necessárias.
